# Incident Response Lab

## Overview
This project simulates real-world security incidents and demonstrates the full incident response lifecycle.

## Incidents Simulated
- Brute Force Login Attack
- Malware Execution
- Suspicious Outbound Traffic

## Tools Used
- Splunk
- Sysmon
- Windows Event Logs
- Kali Linux

## Skills Demonstrated
- Incident Triage
- Timeline Reconstruction
- Root Cause Analysis
- Log Analysis (SIEM)
- Evidence Documentation
- IR Reporting

## Incident Response Lifecycle
Each incident follows:
Detect → Contain → Eradicate → Recover

## Key Deliverables
- Incident timelines
- Evidence screenshots
- Technical IR reports
- Executive summary
- NIST/ISO compliance mapping
